Title: ToySerialController DancerFork

Tag line: MMD and dancer-focused ToySerialController fork with Virtual Reference motion support.

Description:
ToySerialController DancerFork is a dancer-oriented fork of the original ToySerialController plugin for VaM.

This fork focuses on MMD and dance-heavy workflows and adds a Virtual Reference motion source designed for virtual insertion control. The Virtual Reference system adds configurable Offset0 and Offset1 behavior, follow controls, spring-damper motion, and debugging markers so users can tune how a virtual reference tracks and reacts to a target point.

Key points:
- Adds Virtual Reference motion support for dancer and MMD workflows
- Keeps the original ToySerialController foundation and credits visible
- Designed for users who want more controllable virtual motion than a direct body-part reference alone
